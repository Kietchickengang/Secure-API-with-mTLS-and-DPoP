# Secure-API-with-mTLS-and-DPoP
Zero-Trust API gateway using mTLS for M2M and DPoP/PoP for user authentication. Ensures authenticity, integrity, and replay protection for cloud-native APIs, with support for certificate lifecycle, key rotation, and revocation.

# How to use?
* Step 1: git clone <URL>
* Step 2: cd to folder that involves docker-compose.yml
* Step 3: Open your terminal
* Step 4: Run command 
```bash
   docker compose up -d --build
```
* Step 5: Relax & enjoy
